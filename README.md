# Favoritos Api

## Acessar Documentação com Swagger

http://localhost:8080/swagger-ui.html

## Endpoints criados

Listar Favoritos - GET -> /favoritos
Exibir Favorito - GET -> /favoritos/id
Criar Favorito - POST -> /favoritos
Atualizar Favorito - PUT -> /favoritos/id
Excluir Favorito - DELETE -> /favoritos/id
