# Favoritos Api

## Acessar Documentação com Swagger

http://localhost:8080/swagger-ui.html

## Endpoints criados

- GET/ Listar Favoritos -> /favoritos
- GET/ Exibir Favorito -> /favoritos/id
- POST/ Criar Favorito -> /favoritos
- PUT/ Atualizar Favorito -> /favoritos/id
- DELETE/ Excluir Favorito -> /favoritos/id
